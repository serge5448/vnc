This version of VNC is not compatible with any other versions of VNC.
It has been customized to hide protocol details and to customize password encryption key.

To customize:
-------------
change d3desObfuscationKey array in Password.cxx file

To build:
---------
Open vnc.sln file in VS2010.  Wait until VS parsing step is complete.
Select Release build. Clean Solution/Rebuild Solution.

Output will be in win\Release subdirectory:

vncconfig.exe - change password, configure ports, disable Java port, enable listening port: 80 or 443
vncviewer.exe - client
vncserver.exe - server

This version works over any port (80, 443).
